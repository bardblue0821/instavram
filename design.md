# Instavram 設計
## 概要
- ツイッター・インスタグラムライクにする

## 第一段階
- タイムライン
  - 画像拡大
  - リアルタイム更新
  - フレンド同士しか投稿を見れない
- 画像投稿
  - 画像を6つまで
  - テキストも投稿可能
- フォロー/フォロー解除
- ブロック/ブロック解除 
- プロフィール
  - ヘッダー
  - アイコン（）
  - 自己紹介文（VRChat）
  - 投稿画像一覧
  - フレンド一覧

## 第二段階
- 画像投稿
  - 撮影場所を設定可能
    - VRChat API を使用して、ワールド検索を可能に
    - VRChat API を使用して、フレンドの紐づけを可能に
  - ここじゃない？投稿
  - 私もいたよ！/私はいないよ投稿
  - リプライ機能
- お気に入り機能
- リポスト機能
- プロフィール
  - アルバム機能
  - オンライン状況
  -
